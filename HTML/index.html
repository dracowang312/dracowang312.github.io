HTML

发表于 2024-03-10 更新于 2024-06-13 分类于 前端

HTML（Hypertext Markup Language）是一种用于创建网页的标记语言。它由一系列标签组成，这些标签描述了网页上的各种元素，如标题、段落、链接、图像等。HTML的主要目的是结构化文档并向浏览器传达如何显示文档内容的信息。

网页的基本结构

声明

是H5的声明位于文档的最前面，处于标签之前。 他是网页必备的组成部分，避免浏览器的怪异模式。

基础标签

html标签

<html>定义 HTML 文档，这个元素我们浏览器看到后就明白这是个HTML文档了，所以你的其它元素要包裹在它里面，标签限定了文档的开始点和结束点。

head标签

<head>标签用于定义文档的头部。文档的头部描述了文档的各种属性和信息，包括文档的标题、在 Web 中的位置以及和其他文档的关系等。绝大多数文档头部包含的数据都不会真正作为内容显示给读者。
head标签必须包含在html标签中

meta标签

<meta>叫元信息/元数据标签
提供有关页面的元信息, 比如针对搜索引擎和更新频度的描述和关键词,meta标签用来描述一个HTML网页文档的属性，关键词等

<meta>标签的属性 值 描述
charset UTF-8 文档字符编码
name charset、http-equiv、viewport 文档级元数据的名称
content 根据name的取值设置相应的值

title标签

<title>

  1. 可定义文档的标题。
  2. 它显示在浏览器窗口的标题栏或状态栏上。
  3. <title> 标签是 <head>标签中唯一必须要求包含的东西，就是说写head一定要写title
  4. <title>的增加有利于SEO优化

SEO是搜索引擎优化的英文缩写。通过对网站内容调整，满足搜索引擎的排名需求

body标签

<body>标签定义文档的主体。
body 元素包含文档的所有内容（比如文本、超链接、图像、表格和列表等等。）
它会直接在页面中显示出来,也就是用户可以直观看到的内容

body 元素有且只有一个

标题标签

一个html文件中只能有一个h1标签，合理使用标签利于SEO

快捷键h$*6快速生成h1-h6六个标签

** 正确使用标题标签 **

  • 请确保将 HTML 标题标签只用于标题。
  • 不要为了让字体变粗变大而使用标题标签
  • 正确使用标题标签有益于ESO

行、块、行内块元素

块元素

**常见的块元素 **

<h1> <h6> <p> <div> <ul> <ol>

**特点 **

  • 自身独占一行
  • 高度、宽度、内外边距都可以自定义
  • 宽度默认是父元素的100%
  • 块元素里可以放行内、行内块、块元素

行块元素

<img> <input> <td>因为它们具有块元素的特点和行内元素的特点，故称为行内块元素。

特点

  • 和相邻的行内元素（包含行内块）在一行上，它们直接会有空白缝隙
  • 一行可以显示多个（行内元素特点）
  • 默认宽度就是内容的宽度（行内元素特点）

注意

段落元素（文字元素）内不能放块元素

例如，p标签里不能放div标签，h1中也不能放div

行元素

<a> <span> <li>

特点

  • 相邻行内元素在一行，一行可以显示多个。
  • 高度、宽度的设置无效，只会被文字撑开。
  • 默认宽度就是文本撑开的长度
  • 行内元素只能容纳行内元素和文本

注意：

超链接里不能容纳超链接

超链接可以放块级元素，但是为了代码工整，最好把a标签转换为块元素

列表

有序列表

有序列表是一列项目，列表项目使用数字进行标记。 有序列表始于<ol>标签。每个列表项始于 <li>标签。

type属性<ol>的属性type拥有的选项

有序标签嵌套

无序列表

无序列表是一个项目的列表，此列项目使用粗体圆点（典型的小黑圆圈）进行标记
无序列表始于 <ul> 标签。每个列表项始于<li> 标签。

<ul>是块元素，<li>是行内元素

常见应用场景

  • 无序标签的列表效果
  • 导航效果

列表标签之自定义列表

自定义列表实现
自定义列表以 <dl> 标签开始。每个自定义列表项以 <dt> 开始。每个自定义列表项的定义以 <dd> 开始。

应用场景
在一些需要对列表进行解释的部分，可以使用自定义列表
