运行程序

要运行创建的 .NET 项目，你需要使用 dotnet run 命令。在 Visual Studio Code 中，打开终端并导航到项目文件夹，然后运行该命令。以下是一些具体步骤，帮助你在 VS Code 中运行不同类型的 .NET 项目。

  1. 控制台应用程序 (Console Application)
  1. ASP.NET Core Web API

运行后，你可以通过浏览器或 Postman 访问默认情况下运行在 http://localhost:5000 或 https://localhost:5001 的 API。

  1. ASP.NET Core MVC 应用程序

运行后，你可以通过浏览器访问默认情况下运行在 http://localhost:5000 或 https://localhost:5001 的应用。

  1. ASP.NET Core 带 Razor Pages 的 Web 应用程序

运行后，你可以通过浏览器访问默认情况下运行在 http://localhost:5000 或 https://localhost:5001 的应用。

  1. 单元测试项目 (Unit Test Project)

运行后，终端会显示测试的执行结果。

  1. WinForms 应用程序 (.NET Core)

运行后，WinForms 应用程序窗口将会启动。

  1. WPF 应用程序 (.NET Core)

运行后，WPF 应用程序窗口将会启动。

  1. gRPC 服务

运行后，gRPC 服务将会启动，默认情况下运行在 http://localhost:5000 或 https://localhost:5001。

  1. Worker 服务

运行后，Worker 服务将会启动并运行在后台。

在 VS Code 中运行项目

  1. 打开项目：在 VS Code 中，点击“文件” -> “打开文件夹”，选择你的项目文件夹。

  2. 打开终端：点击“终端” -> “新终端”，或者按 Ctrl+ `。

  3. 运行项目：在终端中导航到项目文件夹，然后运行 dotnet run 命令。
